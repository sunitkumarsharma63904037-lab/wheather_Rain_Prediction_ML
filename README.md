# wheather_Rain_Prediction_ML

This project is a Machine Learning based Rain Prediction System developed using Python and the Gaussian Naive Bayes algorithm. The model predicts whether rain is expected or not based on three weather parameters: temperature, humidity, and wind speed.

The dataset contains weather records with corresponding rain labels. The data is divided into training and testing sets using train_test_split from scikit-learn. A Gaussian Naive Bayes classifier is trained on the dataset and used to predict rainfall for new weather conditions entered by the user.

Users can provide temperature, humidity, and wind speed values, and the model predicts whether rain is expected or not. The performance of the model is evaluated using accuracy score on the test dataset.

Technologies Used:
Python, NumPy, Pandas, Scikit-learn

Machine Learning Algorithm:
Gaussian Naive Bayes

Features:
• Weather data classification
• Train-test data splitting
• Gaussian Naive Bayes model training
• Real-time rainfall prediction
• Accuracy evaluation
• Beginner-friendly implementation

Input Parameters:
• Temperature
• Humidity
• Wind Speed

Output:
• Rain Expected
• No Rain Expected

This project is designed for students and beginners who want to learn machine learning classification algorithms and practical weather prediction using Python. It demonstrates the complete workflow of data preparation, model training, prediction, and performance evaluation in a simple and easy-to-understand manner.
